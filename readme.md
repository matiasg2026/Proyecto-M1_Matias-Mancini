# 📖 Descripción

Este proyecto fue desarrollado como parte del **Módulo 1** del curso de Desarrollo Web.

La aplicación permite generar paletas de colores aleatorias en formato **HEX** o **HSL**, elegir la cantidad de colores y copiar cada código al portapapeles con un solo clic.

---

# ✨ Funcionalidades

✅ Generación aleatoria de colores

✅ Selección de 6, 8 o 9 colores

✅ Formato HEX

✅ Formato HSL

✅ Copiado del código al portapapeles

✅ Toast de confirmación

# 💻 Cómo utilizar

1. Elegir la cantidad de colores.
2. Seleccionar el formato (HEX o HSL).
3. Presionar **Generar Paleta**.
4. Hacer clic sobre cualquier color para copiar su código.
