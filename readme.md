# Generador de Paletas de Colores

Este proyecto genera paletas de colores en formato HEX o HSL.

## Funcionalidades

-Generar colores aleatorios.
-Elegir cantidad de colores.
-Elegir formato HEX o HSL.
-Copiar color al portapapeles con un click.
-Notificacion tipo toast.
-Permite poder bloquear un color.

## Tecnologias

-HTML
-CSS
-JavaScript

## Como usar

1. Abrir el proyecto.
2. Elegir la cantidad y formato
3. presionar el boton "Generar Paleta".
4. Hacer click en un color para poder copiarlo.
5. Presionar el candado en caso de que se desee bloquear un color.
